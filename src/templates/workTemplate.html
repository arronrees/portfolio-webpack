Pokemon Generator

Click to load Pokemon. Built using HTML, CSS and Javascript. Most of the HTML is generated through Javascript. Uses the Fetch API and Async Await to gather Pokemon data from pokeapi.co. Cycles through available Pokemon and displays their picture, name and abilities to the user.

Character Search

Search for any character in Middle Earth. Built using the Javascript Fetch API and Async Await to gather and present data from the-one-api.dev. Allows the user to input a character name. It will search the API for all matches and present the name of each matched character. The user can then click on their preferred character and details about the character will appear.

Rock, Paper, Scissors

Play rock, paper, scissors against the computer. Built using HTML, CSS and Javascript. Stores the player's name and scores in local storage and will present this on reload if the user enters the same name. Scores can be reset by clicking the button or if a new name is entered.
